# Hobart 2018 Flood Project Handoff

## Vision
Build a Tasmania-focused flood intelligence system that starts with retrospective flood mapping of the May 2018 Hobart flood using Sentinel-1 SAR and evolves into a continuously updated flood-risk monitoring platform.

## Main Goal
1. Detect and map flood extent from Sentinel-1 SAR.
2. Validate outputs using available Tasmanian flood resources.
3. Develop flood susceptibility and risk prediction capabilities.
4. Build an operational system for insurers, emergency management, and local government.

## Current Status
- Sentinel-1 SAR workflow identified.
- Google Earth Engine selected as primary platform.
- Tasmania flood calibration and water-level datasets collected.
- Validation strategy under investigation.
- Continuous monitoring concept defined.

## Key Challenge
The available calibration datasets appear to contain water-level information rather than comprehensive flood-depth or flood-extent observations.

## Success Criteria
- Reliable flood extent extraction.
- Quantified validation metrics.
- Repeatable workflow.
- Operational update capability.
- Commercial applicability.
